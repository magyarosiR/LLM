# Market React Frontend

This project is a React application created with Vite, styled with Ant Design. It connects to a FastAPI backend for product management (list, view, add, edit, delete).

## Features
- Product listing
- Product details
- Add product
- Edit product
- Delete product

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Ensure the FastAPI backend is running and CORS is enabled.

## Backend
The backend is located in the `../03_python_fastapi_project` folder. Make sure to enable CORS in FastAPI for API calls from the frontend.

## Styling
Ant Design is used for a modern, stylish UI.
